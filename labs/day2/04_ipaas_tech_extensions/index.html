Custom Steps

  • Certain client requirements mandate extensions to applications

  • Developers:

    • Create Apache Camel-based project with additional metadata

    • Implement custom step using project

  • Extension can:

    • Consist of multiple steps

    • Be implemented as Java bean or Camel route

  • Step is aware of data types

  • Supported in Red Hat Fuse tooling (Red Hat Developer Studio 11.2+)

Step Extensions

  • Developers create Camel project with additional metadata

  • Power of Camel wrapped into Fuse Ignite step

    • Multiple steps allowed per extension

  • Extensions can be implemented as Camel route or Java bean

  • Aware of data type

  • Use to add custom JDBC drivers

  • Fuse tooling support

  • If not in box, build extension
