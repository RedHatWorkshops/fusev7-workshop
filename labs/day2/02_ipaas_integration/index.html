Managing Integrations

  • Integration portability

    • Support for multiple Fuse Ignite environments: Development, Test, Production

  • Environment portability

    • To and from cloud

    • To and from on-premise

  • Integration export

    • Packages everything needed to run integration within single ZIP archive

    • Connections, extensions, metadata

  • Integration import

    • Uploads integration with all dependencies into Fuse Online environment

  • Update of connections

    • Needed when porting between environments

Shapeless vs. Shaped Connectors

  • Shapeless content: Data type defined at configuration time

  • Shaped content: Data type not defined at configuration time, only discovered at runtime

  • Shapeless connector: Handles shapeless content

  • Shaped connector: Handles shaped content

Connector Types

Twitter Connector

  • Search for tweets that mention you

  • Search for tweets that contain text that you specify

  • Secured via access/secret keys

Connector Types

Salesforce Connector

  • Supports creating, fetching, updating, deleting Salesforce records

    • Uses record ID or external ID

  • Supports receiving notifications of modifications to records

  • Shaped connector

  • Retrieves data schema from Salesforce objects being accessed

  • Secured via access/secret keys

Connector Types

Slack Connector

  • Supports publishing messages to user or channel on Slack collaboration software

  • Supports shaped data

Connector Types

Database Connector

  • JDBC-like syntax for SQL statements

  • Supports periodic and event-driven SQL invocation

  • Data types dynamically generated by SQL

    • Not bulletproof

    • Feedback appreciated

  • Baseline SQL verification to prevent careless errors

  • Built-in driver support for:

    • PostgreSQL

    • MySQL

  • Supports add-your-own driver 

Connector Types

AMQP Connector

  • Compatible with xPaaS AMQ broker images, AMQ Online

  • Cold AMQ instance included in Fuse Online environment for messaging scenarios

  • Supports Subscribe-Publish, Request-Response typologies

  • Shapeless connector

  • Supports SSL/TLS encryption, basic authentication
